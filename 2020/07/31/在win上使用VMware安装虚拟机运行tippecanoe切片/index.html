发表于 2020-07-31

在win上使用VMware安装虚拟机运行tippecanoe切片

记录自己安装 tippecanoe 的过程
tippecanoe 为生成本地矢量瓦片的开源工具，介绍文章在Mapbox GL JS学习笔记四：开源工具生成本地矢量瓦片 、Mapbox GL JS学习笔记七：mb-util工具使用。
步骤为使用 VMware 安装 Linux 虚拟机，将 tippecanoe 部署在虚拟机上。再将启动服务部署在虚拟机上，使用命令调用切片。用另外的 FoxGIS Server Lite 部署在 Windows 主机上来访问切片后的瓦片文件
参考文章：FoxGIS安装部署 、Mapbox GL JS 学习笔记八：tippecanoe在centos上安装以及使用补充说明
在安装过程中出现错误，无法继续在本机安装。

1.安装VMware虚拟机

从官网（试用 VMware Workstation Pro）下载最新版 VMware 。

在http://mirrors.aliyun.com/centos/7/isos/x86_64/CentOS-7-x86_64-Minimal-2003.iso 下载好 iso 镜像文件。

一路下一步完成 VMware 的安装，创建新的虚拟机。使用 centos7 的镜像包完成安装。

2.在centos上安装tippecanoe

安装完后尝试 ping 114 ，发现报错

使用https://blog.csdn.net/yohjob/article/details/90724914 这个博客的方法解决了。

2.1.解决centos系统网络不通问题

  1. 进入网卡配置文件所在目录：

  2. 编辑网卡配置文件

  3. 将文件中的最后一行的ONBOOT=no 改为ONBOOT=yes

  4. 保存文件并退出vi，重启network

2.2.安装VMtool

这个工具安装一下还是很方便的，可以操作的更加流畅。

  1. 选择 虚拟机 >> 客户机 >> 安装/升级vmwareTools
  2. 装载cd(挂载到media文件夹下)：
  3. 如果出现bash:./vmware-install.pl :/usr/bin/perl:bad interpreter:No such file or directory，输入yum groupinstall "Perl Support"即可。
  4. 一路回车，出现重复现象时注意翻译一下后选择yes或no

2.3.安装tippecanoe

  1. 先更新依赖

  2. 挂载虚拟机
    将虚拟机关机后，选择编辑虚拟机设置。

在虚拟机设置中选择选项->共享文件夹

选择下一步

填写文件夹路径

完成

  1. 出现问题
    用 VMware 安装的 Centos7 进行到这里时无法共享文件夹，参考同事的方法用 virtualBox 安装的Centos7 无法开机。所以放弃这种方式运行。
